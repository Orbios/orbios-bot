# Orbios AI Bot

A Discord bot that automatically translates messages and supports slash commands for additional functionality.

## Features

- Automatic message translation (Thai/Russian â†’ English)
- Slash commands support:
  - `/call` - Create a video call link (with private option)
  - `/ping` - Simple ping command for testing

## Available Scripts

- `npm run dev` - Run the bot in development mode with hot reload
- `npm run deploy-commands` - Deploy/update slash commands
- `npm run build` - Build the TypeScript code
- `npm run start` - Run the bot in production mode

## Command Usage

### Translation

The bot automatically translates messages in configured channels from Thai or Russian to English.

### Slash Commands

- `/call` - Create a video call link
- `/ping` - Test bot responsiveness
